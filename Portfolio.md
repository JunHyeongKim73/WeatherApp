# Project
React-native & Expo를 이용한 첫 번째 프로젝트이다.  
Nomad coder의 "React native로 날씨앱 만들기"를 참고하여 만들었다.  
앱을 실행하면 OpenWeatherMap의 API를 통해 현재 사용자의 위치 정보를 파악하고  
그 위치의 날씨 정보를 바탕으로 현재 기온과 날씨를 알려준다.  
기온과 날씨에 따라 디자인을 달리하여 기온과 날씨에 대한 정보를 시각적으로 전달하였다.  
# Learned
React-native, Expo의 기본적인 사용법에 대해 알 수 있었고  
Open API & API KEY를 통해 데이터를 가져오는 방식에 대해 알 수 있었다.  
API를 사용하는 것이 생각보다 어렵지 않다는 것을 알게 되었고   
다양한 API를 통해 앱을 만들어보고 싶어졌다.
